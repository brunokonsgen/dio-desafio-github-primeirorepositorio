Aula 1 - Estrutura básica

Nessa aula aprendi que HTML (HyperText Markup Language) não é uma linguagem de programação. É uma linguagem de marcação utilizada para dizer ao seu navegador como estruturar a página web que você visita. A página pode ser tanto complicada como simples quanto o desenvolvedor web desejar que seja. Existem comandos básicos para a inicialização de uma página em HTML. O html é o comando inicial onde todos os demais estarão contidos nele.Já o head e body são o cabeçalho e corpo do site respectivamente.

Aula 2 - Tags no HTML

Aqui foram mostradas as principais tags utilizadas nos comandos mais básicos em HTML. As tags são sempre iniciadas por um sinal de menor e fechadas com o sinal de maior. Existem vários tipos e cada uma assume uma função diferente na sintaxe do HTML.

Aula 3 - Textos
